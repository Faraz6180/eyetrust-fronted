# EyeTrust AI — Frontend

AI-powered eye disease detection app (Vite, vanilla HTML/CSS/JS).

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production Build

```bash
npm run build
```

## Environment Variables

Copy `.env.example` to `.env` and set your backend API URL:

```
VITE_API_BASE_URL=https://faraz618-eyetrust-backend.hf.space/api
```

If not set, it defaults to the EyeTrust backend on Hugging Face Spaces.
